Example Repo
Description
This repository contains a simple program demonstrating basic Git version control and code management.

Usage
Run the program:

create the correct file
echo 'print("Hello, World!")' > hello_world.py

or if JavaScript
echo 'console.log("Hello, World!");' > hello_world.js

stage and commit
git add hello_world.py git commit -m "Add hello world program"

python hello_world.py

push
git push origin main
